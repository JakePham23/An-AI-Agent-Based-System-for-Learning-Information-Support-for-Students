# COURSE: Hệ quản trị cơ sở dữ liệu (CSC12003)

## GENERAL INFORMATION
- **Course ID**: CSC12003
- **English Name**: Database Management Systems
- **Vietnamese Name**: Hệ quản trị cơ sở dữ liệu
- **Responsible Person**: for the module: Dr. Phạm Thị Bạch Huệ
- **Semester**: (s) in which the module is taught: 5
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

- Conceptual Database Design (part 1)
- Conceptual Database Design (part 2)
- Data Dependency & Normal Form
- Logic Database Design
- Physical Database Design (part 1)
- Physical Database Design (part 2)
- Advanced Topics
- Review

## LEARNING OBJECTIVES (CLOs)
- G1.Apply soft skills independently or in group to apply principles of
RDBMSs
- G2.Practice the reading comprehension skills, present and write simple reports in English
- G3.Use the functions of a RDBMS, analysis and design to solve real-world information systems
- G4.Describe the general architecture of a DBMS
- G5.Operate  on  a  relational  database  consistently  using transaction processing concepts and theory
- G6.Use concurrency control techniques provided by DBMSs
- G7.Explain the techniques used in DBMSs for database recovery from failures
- G8.Applyi security mechanisms provided by DBMSs in real- world applications
- G9.Demonstrate the way data is stored or retrieved on storage devices
- G10.Use the query processing algorithms and query optimization methods
- G11.Apply functions provided by SQL Server together with a programming language in  real-world information systems
