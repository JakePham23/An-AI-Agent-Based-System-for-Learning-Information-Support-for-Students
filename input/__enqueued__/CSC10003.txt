# COURSE: Phương pháp lập trình hướng đối tượng (CSC10003)

## GENERAL INFORMATION
- **Course ID**: CSC10003
- **English Name**: Object-Oriented Programming
- **Vietnamese Name**: Phương pháp lập trình hướng đối tượng
- **Responsible Person**: for the module: Dr. Đinh Bá Tiến
- **Semester**: (s) in which the module is taught: 4
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction to Object-Orient Concepts and Programming
- Class, constructors and destructor
- Operator overloading
- Inheritance
- Inheritance and polymorphism
- Const-correctness & Review
- Midterm Exam
- Templates and meta-programming
- Exception handling & RAII
- Standard template library (STL): vector, string, streams
- Design patterns: Singleton, Composite
- Review for the final exam

## LEARNING OBJECTIVES (CLOs)
- G1.Work independently or in groups to solve problems using object-oriented concepts.
- G2.Perform reading comprehension skills, write and present simple reports in English
- G3.Use object-oriented, analysis and design concepts to solve real-world problems.
- G4.Comprehend the OOP concepts: class, object, data hiding, encapsulation, inheritance, dynamic binding...
- G5.Describe the key differences between C++, C#, and Java on some OOP topics
- G6.Solve problems in C++ using OOP
