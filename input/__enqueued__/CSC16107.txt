# COURSE: Ứng dụng thị giác máy tính (CSC16107)

## GENERAL INFORMATION
- **Course ID**: CSC16107
- **English Name**: Applications of Computer Vision
- **Vietnamese Name**: Ứng dụng thị giác máy tính
- **Responsible Person**: for the module: Dr. Võ Hoài Việt
- **Semester**: (s) in which the module is taught: 8
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Data Collection for Image and Video
- Image Template Matching and Application
- Image Segmentation
- Introduce final projects and programming environments
- Image Stereo
- Convolution Neural Network
- Generative Adversarial Network
- Action Recognition
- Activity Recognition
- Guest Lecturer
- Random Field and Application in Computer Vision

## LEARNING OBJECTIVES (CLOs)
- G1.Practice on an individual and groups to present a number of technical reports in the field of computer vision.
- G2.Interpret how  to build software product
- G3.Outline how to start professional activities through course projects, analyze ideas for future products
- G4.Construct a software product in computer vision with regard to concrete requirements
- G5.Practice open source technologies to build a computer vision system
- G6.Recognize the applications of computer vision in industry context
