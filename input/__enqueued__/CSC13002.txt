## LEARNING OBJECTIVES (CLOs)
- G1.Apply personal skills and teamwork skills to present reports based on available templates
- G2.Explain software concepts in English
- G3.Explain basic concepts, principles, methods, and techniques in software engineering
- G4.Classify and compare different software processes
- G5.Set up at basic level each phase in software development lifecycle
- G6.Construct software artifacts in each phases in software process
- G7.Use CASE tools
