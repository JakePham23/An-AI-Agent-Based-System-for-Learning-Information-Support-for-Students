# COURSE: Lập trình song song (CSC14120)

## GENERAL INFORMATION
- **Course ID**: CSC14120
- **English Name**: Parallel Programming
- **Vietnamese Name**: Lập trình song song
- **Responsible Person**: for the module: Trần Trung Kiên
- **Semester**: (s) in which the module is taught: 7
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction to CUDA; example: vector addition, matrix multiplication, convolution, … (2 weeks)
- GPU parallel execution in CUDA; example: reduction, vector addition, … (3 weeks)
- Types of GPU memories in CUDA; example: reduction, convolution, … (2 weeks)
- Example: scan, histogram, sort (3 weeks)
- Optimizing a CUDA program; additional topics in parallel programming; final project (1 weeks)

## LEARNING OBJECTIVES (CLOs)
- G1.Operate common tasks (vector addition, matrix multiplication, convolution, reduction, scan, histogram, sort, ...) for parallel execution on the GPU using CUDA.
- G2.Apply the knowledge of GPU parallel execution in CUDA to speed up a CUDA program.
- G3.Apply the knowledge of GPU memories in CUDA to speed up a CUDA program.
- G4.Apply the optimization process to optimize a CUDA program.
- G5.Apply teamwork skills to complete the final project.
• Content:
- Introduction to CUDA; example: vector addition, matrix multiplication, convolution, … (2 weeks)
- GPU parallel execution in CUDA; example: reduction, vector addition, … (3 weeks)
- Types of GPU memories in CUDA; example: reduction, convolution, … (2 weeks)
- Example: scan, histogram, sort (3 weeks)
- Optimizing a CUDA program; additional topics in parallel programming; final project (1 weeks)
