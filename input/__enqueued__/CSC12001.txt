# COURSE: An toàn và bảo mật dữ liệu trong hệ thống thông tin (CSC12001)

## GENERAL INFORMATION
- **Course ID**: CSC12001
- **English Name**: Data Security in Information Systems
- **Vietnamese Name**: An toàn và bảo mật dữ liệu trong hệ thống thông tin
- **Responsible Person**: for the module: Dr. Phạm Thị Bạch Huệ
- **Semester**: (s) in which the module is taught: 6
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction to the course
- Chapter 01: An overview
- Chapter 02: User Authentication
- Chapter 03: Access Control (Basic concepts - DAC - RBAC)
- Chapter 03: Access Control (VPD – MAC - OLS)
- Chapter 03: Access Control (OLS)
- Working in group on the given project
- Chapter 04: Database Encryption
- Chapter 05: SQL Injection
- Chapter 06: Auditing
- Chapter 07: Security Issues in ODBS
- Review

## LEARNING OBJECTIVES (CLOs)
- G1.Apply soft skills to independently or in group to apply security mechanisms of RDBMSs
- G2.Practice the reading comprehension skills, and rewrite simple reports in English.
- G3.Explain basic concepts in information security
- G4.Recognize the security requirements in an information system.
- G5.Explain the principles of security schemes or mechanisms provided by RDBMSs for enforcing the security requirements in real-world information systems
- G6.Apply security mechanisms provided by DBMSs in real-world information systems.
- G7.Illustrate  SQL Injection attacks and operate defense methods.
- G8.Explain security issues in ODBS (Outsourced Database Services) and apply security policies in an outsourced database.
