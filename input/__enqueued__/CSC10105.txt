# COURSE: Nhập môn tư duy thuật toán (CSC10105)

## GENERAL INFORMATION
- **Course ID**: CSC10105
- **English Name**: Introduction to Algorithmic Thinking
- **Vietnamese Name**: Nhập môn tư duy thuật toán
- **Responsible Person**: for the module: Dr. Nguyễn Thanh Hùng
- **Semester**: (s) in which the module is taught: 5
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
During the course, students are also trained in their coding skills, their thinking mindset, and coding interviewing skills such as algorithm explanation, complexities optimization, etc.
• COURSE TOPICS:
- Introduction to Competitive Programming, Algorithms and Complexity
- Review variations of simple data structures.
- Introduction of advanced Data Structures
- Recursion and Dynamic Programming Algorithms
- String Processing and Greedy Algorithms
- Mathematical Algorithms and Complexity Optimization
- Geometric Computation
- Introduction to Graph Theory
- Introduction to Game Theory
- Coding Interviewing Process Introduction and Simulation
- Review

## LEARNING OBJECTIVES (CLOs)
- G1.Understanding complexities, in terms of space and time.
- G2.Learning essential data structures.
- G3.Studying a variety of algorithms in many topics related to competitive programming and coding interview questions.
- G4.Training and practicing to gain problem analyzing, explaining, optimizing skill sets.
- G5.Getting used to coding interviewing processes.
- G6.Experimenting coding interviewing processes.
