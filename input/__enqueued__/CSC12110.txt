## COURSE DESCRIPTION
The course helps students understand a data ecosystem and its components, as well as the fundamentals of data analysis such as data collection or data mining. Students also gain the technical and soft skills required to have effective communication of the data to stakeholders, and master these skills. These can provide students with the option to become data-driven decision-making, through exploring Big Data platforms such as Hadoop, Hive, and Spark. The course also helps students explore the fundamentals of data collection and identifying data sources, and how to clean, analyze, and share their data using visualization tools and overview pages.
• COURSE TOPICS:
- Introduce data analytics, roles and responsibilities, data analytics applications
- Introduce the data ecosystem, data structures, data formats, data storage
- Techniques related to Data Marts, Data Lakes, ETL, and Data Pipelines
- Correct exercises related to Data Marts, Data Lakes, ETL, and Data Pipelines
- Introduce big data, basic use of big data processing tools
- Experience the stages of data collection, data entry
- Technique to clean data
- Statistical analysis, data mining and tools, communication and sharing of analytical data
- Statistical analysis, data mining and tools, communication and sharing of analytical data (cont.)
- Visualizing Data and practicing some analysis tools

## LEARNING OBJECTIVES (CLOs)
- G1.Use soft skills to work individual or in collaboration to effectively present technical reports according to pre-defined templates of the course
- G2.Use the key terminologies in English relating to data analysis
- G3.Explain basic concepts, responsibilities, related works, and basic ethical principles to the data analysis.
- G4.Classify the different types of data analysis and the key steps in the data analysis process, and the roles and responsibilities involved.
- G5.Demonstrate data ecosystems, the different types of data structures, data formats, data sources, and data processing languages.
- G6.Execute techniques and tools for data identification, collection, cleaning, and processing
- G7.Explain different statistical analysis tools and techniques, data visualization
