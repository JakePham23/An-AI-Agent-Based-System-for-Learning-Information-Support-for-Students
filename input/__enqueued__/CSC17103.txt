## COURSE DESCRIPTION
• COURSE TOPICS:
- Introducing Graph Data Mining
- Review knowledge related to graphs and data mining
- Samples in static and dynamic graphs; generate graph.
- Graph indexing and ranking
- Mining graph patterns
- Graph Classification
- Clustering and community discovery
- Link Prediction
- Embedding
- Research topics include deep learning for graphs, graph abstracts, recommendation systems, anomaly detection, large-sized graphs.

## LEARNING OBJECTIVES (CLOs)
- G1.Construct scientific reports and subject-related content in teamwork.
- G2.Explain the English terms used in the field of graph mining.
- G3.Explain the basic concepts and terminology used in the field of graph mining.
- G4.Describe the meaning and application of graph data mining problems to real-world problems.
- G5.Explain how basic algorithms work in graph data mining.
- G6.Construct some components of a small-scale graph data mining system.
- G7.Operate tools and libraries.
