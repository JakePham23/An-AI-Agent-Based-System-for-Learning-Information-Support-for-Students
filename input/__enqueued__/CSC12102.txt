# COURSE: Chuyên đề chọn lọc trong Hệ thống thông tin (CSC12102)

## GENERAL INFORMATION
- **Course ID**: CSC12102
- **English Name**: Special Topics in Information Systems
- **Vietnamese Name**: Chuyên đề chọn lọc trong Hệ thống thông tin
- **Responsible Person**: for the module: Assoc. Prof. Lê Nguyễn Hoài Nam
- **Semester**: (s) in which the module is taught: 8
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction of information retrieval systems
- Basic models of information retrieval systems: Boolean model, Vector model, Probabilistic model
- Advanced models of information retrieval systems: Latent Semantic Indexing
- Language Model
- Evaluation of information retrieval systems
- Implementation of information retrieval systems
- Natural language processing
- Applying natural language processing techniques to information retrieval systems
- Recommender systems
- Memory-based collaborative filtering
- Model-based collaborative filtering
- Content-based Filtering
- Implementation of recommender systems
- Review
