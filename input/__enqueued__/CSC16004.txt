## GENERAL INFORMATION
- **Course ID**: CSC16004
- **English Name**: Computer Vision
- **Vietnamese Name**: Thị giác máy tính
- **Responsible Person**: for the module: Dr. Trần Thái Sơn
- **Semester**: (s) in which the module is taught: 6
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Camera Modeling and Camera Calibration
- Image features, Edge, Measurement
- Project’s introduction
- Corner detection, blob and DoG
- SIFT, HOG features
- Color and global features
- Generic features using deep learning model
- Similarity measurement
- Object detection
- Object tracking
