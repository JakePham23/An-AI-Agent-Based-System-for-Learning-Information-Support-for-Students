## COURSE DESCRIPTION
Students will have the opportunity to apply principles, methods and tools to design of the interface of software applications. Students will also discuss and evaluate user interfaces of existing applications. Through this course, students will obtain knowledge and skills in designing good user interfaces for software applications and evaluating user interface as well as practicing teamwork skills.
• COURSE TOPICS:
- Introduction
- Usability
- Learnability, Visibility, and Efficiency
- Design Process
- Task Analysis
- Prototyping
- Web UI
- Mobile UI
- Graphic Design
- Interaction Styles
- Evaluating Interface Designs

## LEARNING OBJECTIVES (CLOs)
- G1.Pratice working in teams to design the user interface of a software application and report results
- G2.Summarize papers and materials in English related to user interface design
- G3.Analyze and evaluate user interfaces of existing applications
- G4.Explain the importance, basic concepts, and components of user interface
- G5.Explain and apply basic principles of user interface for designing desktop, Web, and mobile applications
- G6.Explain and apply processes, methods, and tools for designing user interfaces effectively
- G6.Practice implementing the user interface of software applications according the design
- G7.Apply programming skills to implement user interfaces of the proposed application
