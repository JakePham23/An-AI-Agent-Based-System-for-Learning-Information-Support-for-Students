# COURSE: Phân tích thống kê dữ liệu nhiều biến (CSC16003)

## GENERAL INFORMATION
- **Course ID**: CSC16003
- **English Name**: Multivariate Statistical Analysis
- **Vietnamese Name**: Phân tích thống kê dữ liệu nhiều biến
- **Responsible Person**: for the module: Assoc. Prof. Lý Quốc Ngọc
- **Semester**: (s) in which the module is taught: 6
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction to MSA
- The fundamental concepts of MSA (sample)
- The fundamental concepts of MSA (populations) (cont.)
- Multivariate Linear Regression (sample)
- Multivariate Linear Regression.(population)
- Principal Component Analysis (sample)
- Principal Component Analysis (population) (cont.)
- Factor Analysis (sample)
- Factor Analysis (population)
- Canonical Correlation Analysis
- Multivariate Clustering
- Multivariate Classification
- Seminar on subject project 1 (Topic: Basic concepts in MSA support statistical analysis of university exam scores.)
- Seminar on subject project 2 (Topic:Applying linear regression model and canonical correlation to support recruitment and sales management)
- Seminar on subject project 3 (Topic:Principal Components Analysis, Data Analysis in Image Representation)
- Seminar on subject project 4 (Topic: Clustering and Classification in CodeBook generation support Image Retrieval and Image Classification.)

## LEARNING OBJECTIVES (CLOs)
- G1.Practice the role of individuals and groups in teamwork to enhance learning about MSA.
- G2.Practice reading books and presenting presentations and paper in English on MSA.
- G3.Analyze problem solving methods related to MSA to improve the effectiveness of the solution.
- G4.Apply and Analyze  the process of designing and implementing a multivariate data analysis method.
- G5.Apply and Analyze  the process of designing and implementing clustering, classification method.
- G6.Propose the process of designing and implementing MSA system in practical applications.
