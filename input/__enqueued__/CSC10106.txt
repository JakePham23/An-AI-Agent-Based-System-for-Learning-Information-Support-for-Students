## LEARNING OBJECTIVES (CLOs)
- G5.Prepare students for advanced research on large-scale data analysis using graph representations.
- G1.Apply algorithms from graph theory to solve practical problems across different domains.
- G2.Recognize problems that can be addressed using graph theory.
- G3.Select appropriate graph representations based on the problem context.
- G4.Provide students with an understanding of the practical uses of graph theory in various fields such as traffic flow management, telephone network construction, and social network data analysis.
- G6.Develop core algorithms for software applications related to navigation, pathfinding, and process optimization
