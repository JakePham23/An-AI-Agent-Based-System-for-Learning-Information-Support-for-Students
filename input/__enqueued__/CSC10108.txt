# COURSE: Trực quan hóa dữ liệu (CSC10108)

## GENERAL INFORMATION
- **Course ID**: CSC10108
- **English Name**: Data Visualization
- **Vietnamese Name**: Trực quan hóa dữ liệu
- **Responsible Person**: for the module: Dr. Bùi Tiến Lên
- **Semester**: (s) in which the module is taught: 8
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction to data visualization
- Data types and visual variables and mapping
- Principles of data visualization
- Analysis of components in visualization
- Data analysis based on visualization
- Multidimensional data visualization
- Interactive data visualization
- Graphic design in data visualization
- Evaluation in data visualization
- Final Project and Review

## LEARNING OBJECTIVES (CLOs)
- G1.Conduct data analysis and visual design in cooperation with team members.
- G2.Use English documents related to data visualization skills.
- G3.Practice critical thinking while working on data visualization tasks.
- G4.Use data analysis methods and data visualization tools to visually represent a variety of data types.
- G5.Evaluate the meaning of visualization.
• Content:
- Introduction to data visualization
- Data types and visual variables and mapping
- Principles of data visualization
- Analysis of components in visualization
- Data analysis based on visualization
- Multidimensional data visualization

- Evaluation in data visualization
- Final Project and Review
