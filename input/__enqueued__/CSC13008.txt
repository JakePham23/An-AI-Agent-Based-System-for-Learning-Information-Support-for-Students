# COURSE: Phát triển ứng dụng web (CSC13008)

## GENERAL INFORMATION
- **Course ID**: CSC13008
- **English Name**: Web Application Development
- **Vietnamese Name**: Phát triển ứng dụng web
- **Responsible Person**: for the module: Trần Thị Bích Hạnh
- **Semester**: (s) in which the module is taught: 5
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Course Introduction: Course Objectives, Course Information, Exercises, Project and Grading System
- Introduction to Web & Internet
- Web Design Overview
- Building webpages using HTML
- Building Web Forms using HTML Form
- Adding styles to website using CSS
- Building webpages using CSS Layout
- Client-side programming using JavaScript
- DOM & AJAX
- Introduction to NodeJS
- Server-side programming with NodeJS Express
- Working with Template Engine
- Working with Database: Building basic functions with database: Create, Read, Update & Delete
- Working with Database: Building advanced functinos with database: Browse, Filter, Pagination
- State Management
- Other web technologies

## LEARNING OBJECTIVES (CLOs)
- G1.Apply personal skills or teamwork skills to develop a real web application
- G2.Practice self-learning, reading comprehension English document and making presentation about web development subjects
- G3.Conduct a presentation about the overview of the operation mechanism of web applications, web application models and related technologies
- G4.Practice analyzing, developing and deploying a complete small to medium-sized web application
- G5.Design static web interfaces using HTML & CSS
- G6.Practice programming Client-side processing using JavaScript
- G7.Explain and practice designing and developing dynamic web applications with database connection using NodeJS
- G8.Use supporting tools for web development
