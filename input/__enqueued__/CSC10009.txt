# COURSE: Hệ thống máy tính (CSC10009)

## GENERAL INFORMATION
- **Course ID**: CSC10009
- **English Name**: Computer Systems
- **Vietnamese Name**: Hệ thống máy tính
- **Responsible Person**: for the module: Chung Thuỳ Linh
- **Semester**: (s) in which the module is taught: 4
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 25 hours Hours for laboratory session: ~ 0 hours Hours for self-study, examination preparation, other activities: ~ 57.5 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Overview of computer knowledge, data represented on the computer
- Microprocessor’s organization and operation
- MIPS-32 bit architecture
- Basic MIPS-32 bit implementation
- X86-32 bit architecture
- (Extend IA32 to 64 bits)
- Running program on a system
- Circuit logic: design combination circuit, ALU design way, application of sequence sequence
- Memory Hierarchy
- I/O System
- Review

## LEARNING OBJECTIVES (CLOs)
- G1.Use the specialized English terminology about information technology
- G2.Identify basic machine organization, including processors, memory hierarchical, and input/output architecture
- G3.Translate bit strings to numbers using unsigned, 2’s complement, and IEEE standard floating-point representation system
- G4.Apply instruction set architectures: RISC vs CISC, especially MIPS-32bit & x86-32 bit in building an assembly program
- G5.Illustrate digital circuit at logic level by simulation tool (combinational circuit)
