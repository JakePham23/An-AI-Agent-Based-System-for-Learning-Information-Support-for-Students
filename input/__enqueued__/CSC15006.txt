# COURSE: Nhập môn xử lý ngôn ngữ tự nhiên (CSC15006)

## GENERAL INFORMATION
- **Course ID**: CSC15006
- **English Name**: Introduction to Natural Language Processing
- **Vietnamese Name**: Nhập môn xử lý ngôn ngữ tự nhiên
- **Responsible Person**: for the module: Assoc. Prof. Đinh Điền - Nguyễn Hồng Bửu Long
- **Semester**: (s) in which the module is taught: 5
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction to Natural Languages
- An Introduction to Formal Languages: Representation and Grammar of Language
- An Introduction to Formal Languages: Production rules, Derivation trees and syntactic parsing
- Rule-based  approaches: Earley algorithm
- Corpus-based approaches: TBL and POS tagging
- Language models in the spelling correction task
- Hybrid approaches
- Pre-processing in NLP
- Understanding text classification problems based on statistical probability methods
- Apply to the sentiment analysis task
- Morphological analysis
- Syntactic analysis
- Syntactic parsing
- Other NLP applications
- Review

## LEARNING OBJECTIVES (CLOs)
- G1.Understand the basic concepts of natural language processing (NLP), terminologies in natural language, the similarities and differences between English and Vietnamese in terms of automatic language processing.
- G2.Understand basic NLP problems such as morphological analysis, grammar analysis.
- G3.Apply parsing methods according to rule-based approaches and machine learning based approaches.
- G4.Utilize existing NLP toolkits to process texts
- G5.Understand and utilize existing processing blocks in NLP systems
- G6.Apply individual and group working skills to research and solve NLP problems.
- G7.Search and read technical documents in English.
