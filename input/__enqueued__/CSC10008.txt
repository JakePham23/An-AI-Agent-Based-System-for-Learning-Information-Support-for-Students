# COURSE: Mạng máy tính (CSC10008)

## GENERAL INFORMATION
- **Course ID**: CSC10008
- **English Name**: Computer Networking
- **Vietnamese Name**: Mạng máy tính
- **Responsible Person**: for the module: Huỳnh Thuỵ Bảo Trân
- **Semester**: (s) in which the module is taught: 3
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction
- Basic concepts
- History of the Internet
- OSI Model
- TCP/IP Model
- IP address, subneting
- Application Layer
- DHCP, DNS, HTTP, FTP, SMTP, IMAP, POP
- Socket Programming
- Transport Layer
- UDP
- TCP
- Network Layer
- IP protocol
- Routing
- Router
- ICMP
- NAT
- Data Link Layer
- Parity check, checksum, CRC
- TDM,FDM,CDMA, ALOHA,CSMA
- ARP
- Switch
- Ethernet
- Network Media
- Wired networks
- Wireless networks
- Review

## LEARNING OBJECTIVES (CLOs)
- G1.Apply some skills at the individual and the teamwork level to present problems in the Computer Networking
- G2.Use the specialized English terminology in Computer Networking
- G3.Employ and execute at the system level on the number of Computer Networking problems
- G4.Identify and operate the basic services of the Computer Networking.
- G5.Understand and provide solutions to basic network problems. Use basic algorithms learned to solve real problems.
- G6.Conduct to design, deploy and build a number services of small computer networks.
