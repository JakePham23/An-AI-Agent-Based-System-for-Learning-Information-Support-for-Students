## LEARNING OBJECTIVES (CLOs)
- G1.Use Linux commands and Conda to set up the working environment.
- G2.Use Python with built-in libraries to perform a data science process.
- G3.Use Python with the Numpy library to perform a data science process with numerical data.
- G4.Use Python with the Pandas library to perform a data science process with tabular data.
- G5.Use Python with the Matplotlib library to visualize data.
- G6.Use Jupyter Notebook and writing skills to document the whole data science process.
- G7.Use Git, Github, and teamwork skills to control versions and collaborate with others.
