# COURSE: Cấu trúc dữ liệu và giải thuật (CSC10004)

## GENERAL INFORMATION
- **Course ID**: CSC10004
- **English Name**: Data Structures and Algorithms
- **Vietnamese Name**: Cấu trúc dữ liệu và giải thuật
- **Responsible Person**: for the module: Lê Ngọc Thành
- **Semester**: (s) in which the module is taught: 3
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction to DSA
- Recursion
- Algorithm efficiency: Big-O
- Search algorithms: Linear search and Binary search
- Sorting algorithms: O(n2) algorithms
- Sorting algorithms: O(nlog2n) algorithms
- Sorting algorithms: O(n) algorithms (Radix Sort)
- Reviews: Linked lists, Stack, Queue and Priority Queue
- Tree structures: General tree, Binary search tree
- Midterm Examination
- Tree structures: B-tree with optional extensions covering Red-black tree, AA tree, 2-3 tree, 2-3-4 tree
- Graph structure: Basic concepts and graph traversals (Breadth-first search, Depth-first search).
- Graph structure: Topological sorting, Minimum spanning tree, and Shortest path
- Hashing, Hash table with collision resolution techniques
- Reviews

## LEARNING OBJECTIVES (CLOs)
- G1.Interpret the fundamental concepts and terminologies of Data structures and Algorithms in the given context
- G2.Analyze the algorithms and data structures in terms of time and space complexities using asymptotic notations
- G3.Choose the algorithms and data structures following the problem requirements (e.g., data size and distribution, a tradeoff between execution time and storage)
- G4.Implement the algorithms and data structures using C/C++ programming language
- G5.Promote personal aptitudes for logical thinking and communication
