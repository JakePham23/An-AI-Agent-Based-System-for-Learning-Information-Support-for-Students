# COURSE: Phát triển ứng dụng web nâng cao (CSC13114)

## GENERAL INFORMATION
- **Course ID**: CSC13114
- **English Name**: Advanced Web Application Development
- **Vietnamese Name**: Phát triển ứng dụng web nâng cao
- **Responsible Person**: for the module: Nguyễn Huy Khánh
- **Semester**: (s) in which the module is taught: 8
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Course introduction, React introduction
- State management in React, React functional component
- React hook, React UI libraries
- RESTful API
- Authentication RESTful API
- React router
- React Context, Redux
- Docker introduction
- GraphQL introduction
- Cloud computing

## LEARNING OBJECTIVES (CLOs)
- G1.Practice working as a sole developer or in team to research and present some new software technology
- G2.Explain some English term in Software Engineering
- G3.Describe Web 2.0
- G4.Practice implementing a Single page application with RESTful webservice by using a specific frontend web library/framework
- G5.Explain and apply web scalability: Cloud computing, NOSQL, …
- G6.Propose and apply suitable software technology to a specific software project
- G7.Practice building software projects based on new software technology
