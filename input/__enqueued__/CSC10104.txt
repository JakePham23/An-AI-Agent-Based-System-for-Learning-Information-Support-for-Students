- **Vietnamese Name**: Quy hoạch tuyến tính

- **Semester**: (s) in which the module is taught: 6

## COURSE DESCRIPTION
The course focuses on introducing students to the concepts, some mathematical models, applied problems and simulations in Python to solve problems related to the foundations of linear algebra and linear programming.
• COURSE TOPICS:
- Introduce to the important subjects, tools:
- About some optimization problems
- Reminds about the basic concepts of linear algebra: vectors, matrices
- Introduction about some practical applications of the linear algebra
- Introduce the simulation by some Python software package as numpy, scipy, simpy, ...
- Introduction to the linear programming problems
- Concept
- Introduction
- Components, terms
- Illustrated problem
- Solution method
- Geometric methods for 2D problems.
- Introduce about simplex algorithm
- Finding the basic feasible solution
- Simplex table and interations
- Optimal sign of simplex method
- Basic problems in linear algebra
- • Deterministic calculation
- • Solve the equation of 2 unknowns
- • Gaussian method
- • Matrix product
- • Solve equations of many unknowns
- Some advanced content on linear programming
- • Big M technique in monomorphic method
- • The concept of duality
- • Computational processing for dual problem
- Review: some questions for the midterm test
- About linear algebra and linear programming
- • PCA method
- • SVM method
- • Solve linear programming problems with a software package based on Python.
- Application of linear programming
- • Integer programming problem
- • Introduce transportation problem: introduction, three ways to find bases, algorithms.
- • Other applications in other problems: graph, game, coding theory, ...
- Review 1: about linear algebra
- Review 2: about linear programming

## LEARNING OBJECTIVES (CLOs)
- G1.Know some terms in the optimization problems, linear algebra, the basic components such as vectors, matrices, ...
Understanding what is the linear programming problem and related concepts.
- G2.Solve basic problems in linear algebra such as determining determinant, Gauss method, matrix product, solving system of equations with many unknowns, ...
Know how to solve the problem of linear programming by geometry, simplex algorithms; Understand and explainable the terms in the linear programming.
- G3.Know some practical applications of linear algebra and simulation using numpy, scipy, simpy software packages.
Know the advanced content of linear programming: the big M technique to find the basis in the simplex algorithm, the concept of duality and relate to the original article.
- G4.Study the advanced contents on linear algebra such as PCA, SVM and solving linear programming problem on Python.
Know how to apply linear programming to problems: integer programming, transportation problems, game theory.
