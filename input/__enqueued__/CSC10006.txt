# COURSE: Cơ sở dữ liệu (CSC10006)

## GENERAL INFORMATION
- **Course ID**: CSC10006
- **English Name**: Introduction to Databases
- **Vietnamese Name**: Cơ sở dữ liệu
- **Responsible Person**: for the module: Dr. Phạm Thị Bạch Huệ
- **Semester**: (s) in which the module is taught: 4
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- An overview of database systems
- Entity relationship model
- Relational data model
- Relational algebra
- Structured Query Language (SQL)
- Relational Calculus
- Functional dependency and Normal forms
- Functional dependency and Normal forms (cont.)
- Integrity constraints
- Integrity constraints (cont)
- Review

## LEARNING OBJECTIVES (CLOs)
- G1.Apply soft skills to work independently or in groups to solve issues in the database systems
- G2.Practice English reading comprehension skills to read textbooks and other documents related to database systems
- G3.Explain basic concepts and characteristics of the database and database systems
- G4.Apply the Entity-Relationship Model and Relational Data Model for data modeling
- G5.Use database languages to manipulate the databases
- G6.Discover the quality of a given database scheme and practice the methods to normalize a database scheme
- G7.Describe and define integrity constraints in a database
- G8.Describe the future trends in database systems research
- G9.Set up and manipulate the database using a relational database management system
