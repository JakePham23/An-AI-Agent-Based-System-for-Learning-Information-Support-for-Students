# COURSE: Xử lý ngôn ngữ tự nhiên ứng dụng (CSC15008)

## GENERAL INFORMATION
- **Course ID**: CSC15008
- **English Name**: Natural Language Processing and Applications
- **Vietnamese Name**: Xử lý ngôn ngữ tự nhiên ứng dụng
- **Responsible Person**: for the module: Assoc. Prof. Đinh Điền - Nguyễn Hồng Bửu Long
- **Semester**: (s) in which the module is taught: 6
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction & Problems in NLP
- NLP applications: An introduction
- Language Models
- Text classification
- Text clustering
- Text similarity
- Text summarization
- Text readability
- Machine translation
- Building NLP applications
- Seminar

## LEARNING OBJECTIVES (CLOs)
- G1.Apply team work and individual work skills to build reports on NLP applications
- G2.Understand and explain English terminologies in NLP
- G3.Develop skills in describing, analyzing and modeling a real-world NLP application
- G4.Know and understand metrics to evaluate NLP applications
- G5.Construct some simple NLP applications
- G6.Understand NLP toolkits and libraries
