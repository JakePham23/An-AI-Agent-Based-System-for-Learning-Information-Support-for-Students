## GENERAL INFORMATION
- **Course ID**: CSC12002
- **English Name**: Advanced Database Systems
- **Vietnamese Name**: Cơ sở dữ liệu nâng cao
- **Responsible Person**: for the module: Dr. Nguyễn Trần Minh Thư
- **Semester**: (s) in which the module is taught: 5
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Overview of Relational Database Design
- Data Models: Entity Relationship Model
- Data models: Enhanced Entity Relationship Model Relational Data Model
- Conceptual Database Design (part 1)
- Conceptual Database Design (part 2)
- Data Dependency & Normal Form
- Logic Database Design
- Physical Database Design (part 1)
- Physical Database Design (part 2)
- Advanced Topics
- Review

## LEARNING OBJECTIVES (CLOs)
- G1.Practice teamwork skills
- G2.Practice reading English documents related to relational database design.
- G3.Practice critical and creative thinking in relational database design.
- G4.Explain the meaning and importance of relational databases design
- G5.Practice and applying the basic knowledge for the relational database design phases such as data models, functional dependency, normal form, ...
- G6.Apply some techniques to get user requirements
- G7.Execute the tasks in the conceptual and logical database design.
- G8.Execute the tasks in the physical design phases
- G9.List some modern popular databases currently
