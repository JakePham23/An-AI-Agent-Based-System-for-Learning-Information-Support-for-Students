# COURSE: Phân tích dữ liệu thông minh (CSC17001)

## GENERAL INFORMATION
- **Course ID**: CSC17001
- **English Name**: Intelligent Data Analysis
- **Vietnamese Name**: Phân tích dữ liệu thông minh
- **Responsible Person**: for the module: Dr. Nguyễn Tiến Huy
- **Semester**: (s) in which the module is taught: 8
- **Workload**: (incl. contact hours, self-study hours): Hours for lecture: ~ 37.5 hours Hours for laboratory session: ~ 25 hours Hours for self-study, examination preparation, other activities: ~ 116.25 hours

## COURSE DESCRIPTION
• COURSE TOPICS:
- Introduction of data analysis and machine learning
- Data preprocessing
- Data Visualization
- Principal Component Analysis
- Decision Tree
- Support Vector Machine
- Clustering
- Time series Analysis
- Infectious Disease Modelling
- Review
