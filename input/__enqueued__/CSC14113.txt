## COURSE DESCRIPTION
This course will discuss the major ideas used in the implementation of compilers, including lexical analysis, syntax analysis, semantic analysis, syntax-directed translation, abstract syntax trees, symbol table generation, intermediate code and intermediate code generation, code generation, and runtime systems. As part of the course, students will write a small compiler for a C-like language.
• COURSE TOPICS:
- Introduction to compiler design
- An overview of a simple compiler
- Lexical analysis
- Syntax analysis
- Symbol table generation
- Semantic analysis
- Intermediate code representation
- Intermediate code generation
- Runtime environment
- Target code generation
