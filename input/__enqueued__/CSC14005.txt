- **English Name**: Introduction to Machine Learning

• COURSE TOPICS:

- The feasibility of learning models
- Linear model I
- Error and Noise
- Training and Testing
- VC inequality
- VC dimension
- Bias, Variance and Tradeoff
- Linear Model II
- Neural Network Model
- Overfitting and Underfitting
- Regularization
- Validation
- Support Vector Machine
- Final Project & Review

## LEARNING OBJECTIVES (CLOs)
- G1.Explain the concept of machine learning.
- G2.Explain the reason why machines can learn.
- G3.Produce learning models using high-level languages such as Python or MATLAB.
- G4.Analyze and evaluate results when applying a specific machine learning model to a particular dataset, then propose solutions for improvement.
- G5.Use technical documents related to machine learning in English.
